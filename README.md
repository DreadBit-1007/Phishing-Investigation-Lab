# Phishing-Investigation-Lab
The Phishing Email Analysis & Incident Investigation project is a practical cybersecurity simulation that demonstrates how suspected phishing emails are analyzed using forensic techniques, threat intelligence tools, and structured SOC investigation methods to identify malicious indicators and determine risk.
